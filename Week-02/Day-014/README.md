### Key Concept
<!-- The idea in plain English — no jargon, as if explaining to a junior. -->
Function Calling allows an LLM to recognize when another tool is better suited for a task and request that tool instead of trying to perform the action itself.
An AI agent is an AI system that can decide what actions to take in order to achieve a goal.

### Key Learning
<!-- The one thing you understand now that you didn't before today. -->
I understand that an LLM is not expected to perform every task itself. Instead, its strength is reasoning about when a specialized tool should be used, while the application retains control over executing that action. This delegation is what enables AI systems to evolve from answering questions to accomplishing goals.
