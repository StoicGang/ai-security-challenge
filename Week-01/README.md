# Week 01 — AI Foundations

**Theme:** Python for AI · ML Fundamentals · Neural Networks · Tokenization · Embeddings

---

## Objective
Understand what a model learns, how text becomes numbers, and what a neural network computes — enough to explain it in your own words, not enough to have memorized a library's API.

## Time Budget
Each day targets **~1 hour total**. If a reading resource is running long, stop at the time box and move to Build — you can return to finish reading in the Day 6 buffer.

## Mini Project (Day 5)
By Day 5: retrieve the right paragraph out of 5 given a natural-language query, using library tools. Proves the tokenize → embed → search chain works end to end.
