### Key Concept
<!-- The idea in plain English — no jargon, as if explaining to a junior. -->
1. Models learn through optimization using gradient descent.
2. Neural networks learn useful representations by updating their weights over many training iterations.
3. Text must first be converted into tokens before a neural network can process it.
4. Tokens are transformed into embeddings that capture semantic meaning rather than exact wording.
5. Semantic search retrieves information by comparing embeddings instead of matching keywords.

### How I Approached It
<!-- Numbered steps, briefly. This is the only "steps" section — everything else below should be about the mechanism, not the process. -->
1. Text is first broken into tokens, which are numerical representations the model can process.
2. These tokens are passed through an embedding model that converts them into vectors capturing semantic meaning.
3. Similarity between vectors is computed (e.g., cosine similarity) to retrieve the most relevant information based on meaning rather than exact wording.
